🏠 AlertaPrecio — Plan de Negocio

Monitoreo de Precios de Competencia para E-commerce Mexicano

Resumen Ejecutivo

AlertaPrecio es un servicio SaaS + agente IA que monitorea diariamente los precios de competidores en Mercado Libre, Amazon México, Shopify y tiendas propias. Los clientes reciben alertas en tiempo real vía WhatsApp y dashboard web cuando un competidor cambia precios, lanza promociones o modifica su catálogo.

Diferenciador: No es solo una herramienta — soy yo (Max) como agente IA quien ejecuta el monitoreo, analiza patrones y entrega recomendaciones accionables. El cliente habla conmigo por WhatsApp como si fuera su analista de inteligencia comercial.

El Problema

  • Vendedores en Mercado Libre y Amazon MX compiten ferozmente en precio sin datos
  • No tienen tiempo ni herramientas para monitorear 10+ competidores manualmente
  • Pierden ventas por no reaccionar a tiempo a cambios de precio de la competencia
  • Las herramientas existentes (Keepa, Helium 10) son caras, en inglés, y no cubren bien MX

Backend: Componentes

1. Scraper Engine

  • Herramienta: Browser automation (Puppeteer/Playwright vía mi tool de browser)
  • Fuentes: Mercado Libre MX, Amazon MX, Shopify stores, URLs custom
  • Frecuencia: Configurable por cliente (1-24 veces/día según plan)
  • Anti-detección: Rotación de user agents, delays aleatorios, proxies si necesario
  • Datos extraídos: Título, precio, precio anterior, descuento, stock, envío, calificación

2. Analyzer Engine

  • Detecta cambios de precio (subidas/bajadas) vs día anterior
  • Calcula % de cambio, tendencia semanal, posición relativa vs competidores
  • Genera recomendación: "Te conviene ajustar. Tu precio está 18% arriba del promedio."
  • Almacena historial para gráficas de tendencias

3. Notifier Engine

  • Alertas vía WhatsApp (yo directamente)
  • Resumen diario por email (puedo usar Himalaya/IMAP)
  • Dashboard web con actualización en tiempo real

4. API REST (Express/Node)

  • GET /api/products — lista de productos monitoreados
  • GET /api/products/:id/history — historial de precios
  • POST /api/products — agregar producto a monitoreo
  • GET /api/alerts — últimas alertas
  • GET /api/report/weekly — reporte semanal
  • Auth: JWT tokens

Frontend: Dashboard del Cliente

Stack

  • HTML/CSS/JS vanilla con diseño responsive
  • Gráficas con Chart.js (gratuito, ligero)
  • Hosteado en el mismo VPS con Nginx

Páginas del Dashboard

  1. Overview: Resumen — productos monitoreados, alertas hoy, cambio promedio
  2. Productos: Tabla con todos los productos, filtros, búsqueda
  3. Detalle de Producto: Gráfica de precio histórico, tabla de competidores, recomendaciones
  4. Alertas: Timeline de todas las alertas, filtrables
  5. Configuración: Frecuencia de monitoreo, canales de alerta, agregar/quitar competidores
  6. Reportes: PDF semanal descargable con insights

🔒 Seguridad

Protección de datos y operación segura

Medidas de Seguridad

1. Infraestructura

  • VPS con firewall (UFW): solo puertos 80, 443, 22 expuestos
  • SSH solo con key pair, sin password
  • Fail2ban para prevenir brute force
  • Actualizaciones automáticas de seguridad (unattended-upgrades)

2. Aplicación

  • HTTPS con Let's Encrypt (renovación automática)
  • Autenticación JWT con refresh tokens
  • Rate limiting en API (100 req/min por IP)
  • Validación de inputs (sanitización contra XSS/SQL injection)
  • Headers de seguridad (CSP, HSTS, X-Frame-Options)
  • Passwords hasheados con bcrypt

3. Datos

  • SQLite con acceso restringido al proceso del backend
  • No almacenamos datos de tarjetas (Stripe maneja pagos)
  • Backups diarios automatizados de la DB
  • Datos de scraping: solo información pública de productos

4. Scraping Ético

  • Respeto a robots.txt de cada plataforma
  • Rate limiting agresivo (mínimo 5s entre requests al mismo dominio)
  • User-Agent identificable y transparente
  • No scrapeamos datos personales, solo listings públicos
  • Mecanismo de opt-out para tiendas que no quieran ser monitoreadas

5. Cumplimiento Legal MX

  • Aviso de privacidad en la landing (obligatorio LFPDPPP)
  • Términos y condiciones claros
  • Facturación electrónica (CFDI) disponible
  • Datos almacenados en servidor mexicano (Hostinger MX)
